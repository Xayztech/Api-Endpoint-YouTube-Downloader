The Complete Guide

A comprehensive tutorial on how to use the XAYZ API Ecosystem effectively.

1. Introduction

This system is a Hybrid API designed to fetch direct download links for YouTube videos. It uses a "Failover System" which means if the primary server (Invidious) fails, it automatically switches to a backup server (BotCahx) to ensure you always get a result.

2. Getting Started

Step A: Get an API Key
Go to the API Key Manager menu on the sidebar. Click "Generate Key". Copy that key (starts with xayztech-...). You will need this for every request.
Step B: Using the Home Downloader
1. Paste your API Key.
2. Paste a valid YouTube link (e.g., https://youtu.be/...)
3. Click "Process Media".
4. Wait for the engine to fetch metadata.

3. Using Search Feature

How to Search & Download
1. Navigate to YT Search + DL menu.
2. Enter your keyword (e.g., "Lo-fi Hip Hop").
3. Click Search. The system will retrieve ~10-20 results.
4. Click the "COPY LINK" button on any video card.
5. Scroll down to the "Integrated Downloader" box.
6. Paste the link and click Download.
7. Choose MP4 (Video) or MP3 (Audio) from the buttons.

4. JSON Response Structure

Developers can access the raw data. The structure always looks like this:

{
  "status": true,
  "code": 200,
  "server_used": "Xayz Tech",
  "data": {
    "title": "Video Title",
    "downloads": [
      { "type": "video", "format": "mp4", "url": "..." },
      { "type": "audio", "format": "mp3", "url": "..." }
    ]
  }
}

Node.js Integration

Using axios library. This is the most recommended method.

// 1. Import axios library (npm install axios)
const axios = require('axios');

// 2. Configuration Variables
const BASE_URL = 'https://xayz-ytdl.vercel.app/api/youtube';
const YOUR_API_KEY = 'xayztech-xxxxxxxxxxxxxxxxxxxx'; // Replace with your generated key
const VIDEO_LINK = 'https://www.youtube.com/watch?v=dQw4w9WgXcQ';

// 3. Main Function to Fetch Data
async function getVideoData() {
    try {
        console.log("Fetching data from API...");
        
        // 4. Making the GET Request
        const response = await axios.get(BASE_URL, {
            params: {
                url: VIDEO_LINK,
                apikey: YOUR_API_KEY
            },
            timeout: 60000 // 60s timeout just in case
        });

        const result = response.data;

        // 5. Check if status is true
        if (result.status) {
            console.log("--- SUCCESS ---");
            console.log("Title:", result.data.title);
            console.log("Duration:", result.data.duration);
            console.log("Server Used:", result.server_used);
            
            // 6. Loop through downloads
            console.log("\nDownload Links:");
            result.data.downloads.forEach(item => {
                console.log(`[${item.format.toUpperCase()}] ${item.quality || 'Auto'} -> ${item.url}`);
            });
        } else {
            console.log("API returned error:", result.message);
        }

    } catch (error) {
        // 7. Error Handling
        if (error.response) {
             // Server responded with non-200 code
            console.error("Server Error:", error.response.data);
        } else {
             // Network error
            console.error("Network Error:", error.message);
        }
    }
}

// Execute
getVideoData();

Python Integration

Using requests library. Robust error handling included.

# 1. Import library (pip install requests)
import requests
import json

# 2. Config
API_URL = "https://xayz-ytdl.vercel.app/api/youtube"
API_KEY = "xayztech-xxxxxxxxxxxx"
VIDEO_URL = "https://www.youtube.com/watch?v=dQw4w9WgXcQ"

def download_video():
    print("Connecting to API...")
    
    try:
        # 3. Prepare Parameters
        params = {
            "url": VIDEO_URL,
            "apikey": API_KEY
        }
        
        # 4. Send GET Request
        response = requests.get(API_URL, params=params, timeout=30)
        
        # 5. Check HTTP Status Code
        if response.status_code == 200:
            data = response.json()
            
            if data.get("status") is True:
                print(f"\nTitle: {data['data']['title']}")
                print(f"Author: {data['data']['author']}")
                
                print("\nAvailable Downloads:")
                for item in data['data']['downloads']:
                    quality = item.get('quality', 'Auto')
                    fmt = item.get('format', 'unknown')
                    link = item.get('url')
                    print(f"- [{fmt}] {quality}: {link}")
            else:
                print("API Logic Error:", data.get("message"))
        else:
            print(f"HTTP Error {response.status_code}: {response.text}")
            
    except Exception as e:
        print(f"Critical Error: {str(e)}")

if __name__ == "__main__":
    download_video()
